# JSON Web Token Generator and Validator

JWT is the most popular token method to share secured data between a client and server. This module has JWT generators for JavaScript, PHP, and Python. Also, you can validate the previously generated token with the given server secret. JWT generator uses the SHA-256 algorithm to hash and revalidate the token signature.

Launch Now: [https://deshan.lk/apps/jwt-generator](https://deshan.lk/apps/jwt-generator)

## Key Features

- JSON Web Tokens (JWT) generating, decoding, and validating features.
- Modules for JavaScript, PHP, and Python languages.
